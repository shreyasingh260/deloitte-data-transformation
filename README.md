# Deloitte Data Transformation Task
This project was completed as part of the Deloitte Forage Data Engineering Virtual Internship.

## Project Overview
The task involves transforming JSON data from two different formats into a unified standardized format.

## Features
- Convert device data from Format 1 to the expected structure
- Convert device data from Format 2 to the expected structure
- Handle timestamp conversion from ISO format to milliseconds
- Parse and restructure location data

## Technologies Used
- Python
- JSON
- unittest module

## Files
- main.py – contains the transformation logic
- data-1.json – sample input format 1
- data-2.json – sample input format 2
- data-result.json – expected output format

## Outcome
All test cases pass successfully using Python's unittest framework.
